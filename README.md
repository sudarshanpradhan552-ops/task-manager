# AI Task Manager - Advanced Backend

A high-performance, AI-driven task management system built with FastAPI, SQLModel (MySQL), and OpenAI.

## 🌟 Features

### 🤖 AI Capabilities
- **Smart-Add**: Create tasks using natural language (e.g., "Remind me to call John tomorrow morning").
- **Daily Briefing**: Get a 3-sentence motivational summary of your day's priorities generated by an AI coach.
- **Time Estimation**: AI automatically predicts how many minutes a task will take.
- **Auto-Categorization**: Tasks are automatically grouped into categories (Work, Personal, Health, etc.).
- **Task Decomposition**: Break complex tasks into 3-5 actionable sub-tasks with one click.

### ⚙️ Core System
- **Secure Authentication**: JWT-based login system with encrypted password storage (BCrypt).
- **Real-time Notifications**: Live updates via WebSockets (instant feedback when tasks are created/updated).
- **Persistent Database**: Full CRUD support using SQLModel and SQLite.
- **Multi-user Support**: Each user has their own private workspace.

## 🛠️ Setup Instructions

### 1. Installation
Ensure you have Python 11+ installed, then run:
```bash
pip install -r requirements.txt
```

### 2. Environment Configuration
Create a `.env` file in the root directory and add your credentials:
```env
OPENAI_API_KEY=your_openai_api_key_here
SECRET_KEY=a_long_random_secret_string_for_jwt
```

### 3. Running the Server
```bash
python main.py
```
- **API Docs**: [http://localhost:8000/docs](http://localhost:8000/docs)
- **WebSocket URL**: `ws://localhost:8000/ws/{your_token}`

## 🚀 API Endpoints

| Endpoint | Method | Description |
| :--- | :--- | :--- |
| `/signup` | POST | Create a new user account |
| `/token` | POST | Login and get access token |
| `/tasks/` | GET/POST | List all tasks or create manually |
| `/tasks/smart-add` | POST | Create task using AI natural language |
| `/tasks/briefing` | GET | Generate AI daily summary |
| `/tasks/{id}/decompose`| POST | Break task into AI-generated sub-tasks |
| `/tasks/{id}` | GET/PATCH/DEL| Manage specific task details |
